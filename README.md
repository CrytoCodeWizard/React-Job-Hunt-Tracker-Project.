# ⌨️ Commands

All commands are run from the root of the project, from a terminal:

| Command         | Action                                         |
| :-------------- | :--------------------------------------------- |
| `pnpm install`   | Installs dependencies                          |
| `pnpm start`     | Starts local dev server at `localhost:3000`    |
| `pnpm run build` | Build your production site                     |
